name: concept-reasoner
description: Deep LLM-powered concept explanations with Socratic reasoning, adaptive difficulty, and follow-up Q&A
---

Explains any programming concept using the LLM's reasoning capabilities. Goes beyond static knowledge bases — the LLM generates unique explanations with analogies, code examples, common mistakes, and reasoning chains tailored to the student's level. Supports follow-up questions for deeper understanding.

Usage: "explain [concept] at [level] using concept-reasoner"
