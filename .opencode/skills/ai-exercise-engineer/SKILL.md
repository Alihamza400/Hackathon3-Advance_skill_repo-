---
name: ai-exercise-engineer
description: Generate dynamic AI-powered programming exercises using LLM for any topic and difficulty level
---

Generates fully custom programming exercises via the LLM service (port 8010). Unlike the static template-based exercise service, this creates unique exercises with AI-generated starter code, test cases, hints, and solutions tailored to the student's level.

Usage: "generate [topic] exercises for [level] using ai-exercise-engineer"
