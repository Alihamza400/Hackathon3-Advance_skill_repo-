# AGENTS.md Generation Skill

## When to Use
- Setting up AI agent development in repositories
- Need guidelines for Opencode and Goose
- Creating documentation for agent workflows
- Establishing development conventions

## Instructions
1. Create AGENTS.md template: `./scripts/generate_agents.sh`
2. Customize for repository: `./scripts/customize_agents.sh`
3. Verify template: `./scripts/verify_agents.sh`
4. Deploy to repository: `./scripts/deploy_agents.sh`

## Validation
- [ ] AGENTS.md created with proper structure
- [ ] Repository guidelines documented
- [ ] Agent workflow examples included
- [ ] Development conventions specified

See [REFERENCE.md](./REFERENCE.md) for detailed AGENTS.md format.
